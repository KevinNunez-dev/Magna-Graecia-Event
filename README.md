# 🍋 Magna Graecia Miami — Event Manager

A lightweight, single-file event management PWA built for the **Magna Graecia Miami** dinner event at Donatella Restaurant.

## 🌐 Live App

**[View Live → https://kevinnunez-dev.github.io/Magna-Graecia-Event/](https://kevinnunez-dev.github.io/Magna-Graecia-Event/)**

## 📋 Event Details

- **Date:** Sunday, April 12, 2026
- **Time:** 4:00 PM – 6:00 PM
- **Venue:** Donatella Restaurant, 1350 Collins Ave., Miami Beach, FL 33139
- **Guests:** 50

## 🗂️ Modules

| Tab | Description |
|-----|-------------|
| 📋 Overview | KPI strip, check-in progress, event details |
| 🎟️ Guest List | Live search + check-in with filter chips |
| 🍽️ Menu | Full menu by course + dietary notes |
| 🍷 Wines | Wine list + live bottle counter |
| 🕐 Schedule | Full day timeline |
| 👥 Staff | Staff roles + emergency contacts |
| ✅ Pre-Event | Categorized pre-event checklist |
| 📝 Notes | General notes + post-event debrief |

## 🚀 Usage

Open `index.html` in any modern browser — no server or install required.
